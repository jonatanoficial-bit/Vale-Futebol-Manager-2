# Fluxo de Carreira Definitivo — v7.4.0

## Fluxo final

1. Abrir o jogo.
2. Ir para Central de Slots.
3. Escolher uma carreira existente ou criar uma nova.
4. Entrar no lobby limpo.
5. Jogar, treinar, ver elenco ou abrir menu completo.
6. Usar Salvar e sair para trocar de carreira.

## Problema corrigido

Antes havia uma sensação de telas duplicadas:

Capa → Menu Principal → Jornada Inicial → Lobby.

Agora a decisão principal é simples: escolher o save.
