# Vale Futebol Manager Gold - v3.8.0
## Categorias de base e scouting profundo

Esta build adiciona uma camada de formacao de talentos com:

- tela Base & Scouting no lobby;
- promessas monitoradas;
- prontidao para promocao;
- potencial, risco, valor e personalidade;
- departamentos da academia;
- politica de formacao;
- regioes de scouting;
- eventos narrativos da base;
- template editavel para futuras promessas;
- fallback automatico de fotos.

## Caminho de fotos de jovens

Use o ID do jogador do JSON:

```txt
assets/players/youth/id-do-jogador.png
```

Exemplo:

```txt
assets/players/youth/yan-matheus.png
```

Se a foto nao existir, o jogo usa o placeholder de jogador e nao quebra.
