# v7.3.1 — Hotfix Fluxo Inicial e Slots

Correção solicitada após prints reais do Vercel.

## Problema observado
- Capa e menu principal pareciam duas telas iguais.
- Havia muitos botões técnicos/premium antes do jogador entender o fluxo.
- Não estava claro como sair da carreira para carregar outro save.
- A central de saves existia, mas não estava apresentada como slots jogáveis.

## Correção
- Capa simplificada.
- Menu principal convertido em Central Inicial / Slots.
- Lobby reduzido para ações essenciais.
- Módulos avançados continuam no Menu do Treinador.
- Botão de sair para central inicial e botão de save adicionados ao topo do lobby.
- Criar carreira em Slot Principal, Slot 2 ou Slot 3.
