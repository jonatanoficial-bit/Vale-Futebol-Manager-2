# DEPLOY GUIDE - Vale Futebol Manager Gold Edition v2.4.0

## GitHub Pages
1. Descompacte o ZIP.
2. Envie todos os arquivos para o repositório.
3. Ative GitHub Pages apontando para a branch principal.
4. Abra `index.html` pela URL publicada.

## Vercel
1. Crie um projeto novo na Vercel.
2. Importe o repositório do GitHub.
3. Framework: Other / Static.
4. Build command: vazio.
5. Output directory: raiz do projeto.

## Regras de assets
Substitua imagens nas pastas `assets/` mantendo o mesmo nome e caminho. O código possui fallback automático.
