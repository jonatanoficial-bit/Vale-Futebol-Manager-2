# Vale Futebol Manager - Data Pack 2026.05.20

Build v5.8.0 cria o motor oficial para importação de elencos reais/licenciados travados em 20/05/2026.

## Caminhos oficiais

- Série A: `data/rosters/2026/brazil/serie-a/{clubId}.json`
- Série B: `data/rosters/2026/brazil/serie-b/{clubId}.json`
- Europa: `data/rosters/2026/europe/{leagueId}/{clubId}.json`
- América do Sul: `data/rosters/2026/south-america/{leagueId}/{clubId}.json`
- Seleções: `data/rosters/2026/national-teams/{countryId}.json`

## Regra anti-genérico

Jogadores como `Fluminense GK 1`, `Corinthians ZAG 5`, `Player 1` ou `Jogador 1` são aceitos apenas como template de desenvolvimento. A Release Candidate v6.0.0 deve ficar bloqueada enquanto qualquer clube jogável tiver nomes genéricos.
