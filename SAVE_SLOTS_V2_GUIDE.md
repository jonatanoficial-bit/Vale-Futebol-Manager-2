# Save Slots 2.0 — v7.4.0

Fase 57 do Vale Futebol Manager.

## Objetivo

Consolidar o fluxo inicial do jogo para que o jogador sempre entenda onde está:

Capa → Central de Slots → Criar/Continuar carreira → Lobby limpo → Salvar e sair.

## Slots oficiais

- Carreira principal
- Carreira 2
- Carreira 3
- Carreira 4
- Carreira 5

Cada slot pode ser carregado, salvo, renomeado ou apagado sem misturar com os backups técnicos.

## Regras

- A capa não deve mostrar módulos avançados.
- O menu principal é a central de slots.
- O lobby só mostra ações essenciais.
- Módulos premium entram em Menu completo.
- Sair da carreira salva o slot ativo antes de voltar para a central.
