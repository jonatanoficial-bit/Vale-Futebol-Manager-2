# Vale Futebol Manager — v6.0.0 Beta Pública

Status: public-beta-ready.

## Objetivo
Liberar uma build limpa para teste controlado com jogadores reais, priorizando celular, rolagem por toque, criação de carreira, primeira partida, coletiva, pós-jogo e save.

## Fluxo obrigatório
1. Abrir no celular.
2. Criar carreira.
3. Escolher avatar, nome e clube sem tela cortada.
4. Entrar no lobby.
5. Abrir tutorial.
6. Abrir coletiva.
7. Jogar uma partida.
8. Conferir relatório pós-jogo.
9. Salvar.
10. Fechar e continuar.

## Critério de aprovação
A Beta só avança para v6.1.0 se o jogador conseguir completar a primeira jornada no celular sem travamento ou corte de botão.
